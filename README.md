# Backbonejs-Todo
Backbonejs Todolist rewrite

###简介
* [Backbone.js](http://backbonejs.org/)   
* [Todolist](http://backbonejs.org/examples/todos/index.html)  

安装nodejs，在文件目录下运行`node server.js`，在浏览器输入`localhost:8080`即可访问

代码注释为个人理解，有误请指出，谢谢。

参考学习用。
